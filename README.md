# Телеграм-бот для технической поддержки

Бот создан для интернет-магазина **«Продаем всё на свете»**. Он помогает пользователям найти ответы на часто задаваемые вопросы и перенаправляет сложные запросы специалистам: в отдел **техподдержки** или **отдел продаж**.

---

## Инструменты для реализации:

- Python 3.10+
- [pyTelegramBotAPI (telebot)](https://github.com/eternnoir/pyTelegramBotAPI)
- SQLite3 (для хранения запросов)
- JSON (для хранения FAQ)
- Git/GitHub (для управления версиями)

---

## Проект состоит из:

>  `bot.py` — основной запуск бота, инициализация, команды.  
>  `handlers.py` — логика обработки сообщений и кнопок.  
>  `db.py` — функции для работы с SQLite (добавление заявок, чтение и фильтрация).  
>  `faq.json` — база часто задаваемых вопросов и ответов.  
>  `requirements.txt` — список зависимостей.  
>  `README.md` — описание проекта и инструкция.

---

## Функционал:

1.  Ответы на частые вопросы (FAQ)
2.  Обработка пользовательских сообщений (текст)
3.  Сохранение обращений в БД
4.  Направление запроса в нужный отдел: «Технический» или «Продажи»
5.  Интуитивно понятное меню с кнопками
6.  Документация для администратора и пользователей
